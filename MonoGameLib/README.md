# MonoGameLib
Adds useful members to monogame that help with creating and managing an actual game.

## Features
### Shapes
`IShape` allows for uniform implementation and handling of shapes.
`IPolygon` defines a polygon and simplifies distance and intersection logic across them.

### TextureManager
A central class that streamlines the process of creating, managing and disposing of Texture2Ds.

## Prerequisites
- [.NET SDK](https://dotnet.microsoft.com/download) (version specified in `global.json`)
- [MonoGame](https://www.monogame.net/downloads/) framework
